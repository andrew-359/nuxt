# E2E тесты

- **flows.spec.ts** — сценарий «отмена изменений в редакторе»: ensureAccount, переход на новую заметку, ввод текста, кнопка «Отмена» → в модалке «Отменить изменения» → проверка URL и закрытия модалки.

Запуск: `npm run test:e2e`. С уже запущенным приложением: `E2E_USE_EXISTING_SERVER=1 npm run test:e2e`.
