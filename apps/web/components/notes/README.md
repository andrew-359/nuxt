# Notes components

Компоненты для списка заметок и редактора. Регистрация по имени файла (`pathPrefix: false`).

- **list/** — страница списка: NoteList, NotesListToolbar
- **card/** — карточка заметки в списке: NoteCard, NoteCardTitle, NoteCardTodosPreview, NoteCardTodoRow
- **editor/** — страница редактора заметки: NoteEditorToolbar, NoteEditorForm, TodoItemRow
