# UI components

Обёртки над Nuxt UI. Регистрация по имени файла (`pathPrefix: false`), подпапки — только для структуры.

- **form/** — поля и обёртки форм: AppInput, AppCheckbox, AppSelect, AppFormField
- **card/** — карточки: AppCard, AppPageCard
- **modal/** — модалка: AppModal
- **button/** — кнопки: AppButton, AscDescToggle
- **feedback/** — состояние и пагинация: StateMessage, AppPagination
