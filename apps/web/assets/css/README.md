# Стили приложения

- **`main.css`** — точка входа: Tailwind, Nuxt UI.
- **`app.scss`** — точка входа своих стилей: подключает конфиг и модули.
- **`_config.scss`** — декларативный конфиг (map): font-scale, card.bg.light/dark. Меняй только данные здесь.
- **`_cards.scss`** — переопределение карточек: читает конфиг, выводит `--app-card-bg` и класс `.app-card-bg`.
- Верстка и компоненты — через **Tailwind** и **Nuxt UI**.
